📸 封面图片说明

临时方案：如果您暂时没有准备封面图片，系统会自动显示一个文字占位符。

自定义封面：您可以将您的封面图片命名为 worship-cover.jpg 并放在 icons/ 目录下。

推荐尺寸：400x240像素，长宽比 5:3，格式 JPG 或 PNG

建议内容：教会logo、敬拜主题图像、或者相关的宗教符号

🎼
有光敬拜队歌曲库

如何使用这个默认设计：

  1. 右键点击上方的蓝色封面区域
  2. 选择"另存为图片"或使用截图工具
  3. 保存为 worship-cover.jpg
  4. 放入项目的 icons/ 目录

注意：这只是一个临时的设计方案，您可以随时替换为您自己的封面图片。
