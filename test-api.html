如果看到错误，请按以下步骤操作：

  1. SONG_BUCKET binding is missing
    → 需要在 Cloudflare Pages 设置中配置 R2 绑定
    → 转到 Pages > Settings > Functions > R2 bucket bindings
    → 添加绑定：Variable name: SONG_BUCKET，R2 bucket: worship
  2. 404 错误
    → Pages Functions 未正确部署
    → 确认 functions/api/songs.js 文件存在
  3. Could not list songs
    → R2 存储桶问题
    → 检查存储桶名称和文件结构
